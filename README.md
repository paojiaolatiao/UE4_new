# UE4_new

## 第二次作业

### 作业内容：
在提供案例的基础上添加会飞的物体，命中加分
设置游戏时间为两分钟，在时间结束后通过UI界面展示分数表
### 具体实现：
- 添加飞盘作为会飞的物体（抛物线运动），支持网络同步，飞行物体一直围着一个中心旋转。

- 添加倒计时UI功能：显示当前对局剩余时间。

- 命中飞盘发射者得分加一。

- 对局剩余时间为0，弹出结算UI显示得分。
